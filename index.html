안녕하세요.
슬의 커미션 페이지 입니다.

당분간은 웹 디자인 / 홈페이지 제작 커미션만 받고 있습니다.
이 사이트는 구글 크롬에 최적화 되어있습니다.

종류 Type

  • 원 페이지 - 정적 사이트

    - html / css (js/jquery 미포함)
    - 원페이지 구성 (최대 페이지 3장)
    - github page를 이용한 호스팅(무료)
    - 최대 일주일의 빠른 마감

  • 여러 페이지 - 정적 사이트

    - html / css (js/jquery 미포함)
    - 페이지 구성 (기본 페이지 8개)
    - github page를 이용한 호스팅(무료)
    - 포스팅 솔루션 제공 가능

  • 원 페이지 - 동적 사이트

    - html / css /jquery
    - 원페이지 구성 (최대 페이지 3장)
    - 호스팅 별도(솔루션 제시 가능)
    - 모달, 팝업 등 각종 효과 및 기능

  • 여러 페이지 - 동적 사이트

    - html / css /jquery
    - 페이지 구성 (기본 페이지 8개)
    - 호스팅 별도(솔루션 제시 가능)
    - 모달, 팝업 등 각종 효과 및 기능
    - 텍스트 에디터 (summernote 등)

    • (자세한 사항은 문의 및 견적 후에 알려드립니다.)

작업 순서 process

  • 1
    신청을 받습니다.
    양식 및 연락처는 하단을 참고해주세요.
  • 2
    견적을 낸 뒤 확정하시면 계좌번호를 보내드립니다.
  • 3
    입금 다음날부터 최대 3일 내에 시안을 보내드립니다.
  • 4
    제작 방향에 대한 피드백, 들어가야 할 컨텐츠 등을 받습니다.
  • 5
    최초 견적 날짜의 4/5 경과 이전에 진행 상황을 보여드립니다.
  • 6
    마지막 피드백을 1회 받습니다.
    이 이후엔 더이상 피드백을 받지 않습니다.
  • 7
    최대 5일 내에 최종 결과물을 보내드립니다.
  • ※ 최장 30일을 기준으로 작업합니다.

유의 사항 Notice

타인의 작업물 및 커미션 결과 등을 예시로 들지 말아주세요.
이 경우 작업을 반려할 수 있습니다.

본 커미션은 개인적 / 사적 용도를 위해서만 사용해주세요.
그 외 상업적 등의 이유로 신청하신 뒤 사용하실 수 없습니다.

무단 전제 및 복제, 변경 등을 금합니다. 신청하셨던 목적으로만 사용해주세요.

작업 순서 3번 이후에 타입 변경은 불가능 합니다.
신중히 골라주세요.

작업 순서 3번 이전 커미션 취소 시 전액 환불합니다.
작업 순서 5번 이후 커미션 취소 시 금액의 1/2 환불합니다.
작업 순서 6번 이후 커미션 취소 시 금액의 1/3 환불합니다.
최종 결과물을 전송한 뒤에는 환불하지 않습니다.
